LEARNING LIFE

Thu July 19 2018

今天暂时学完了Coursera的课程“HTML, CSS, and Javascript for Web Developers"，这门课由Johns Hopkins University的Yaakov Chaikin教授讲授，介绍了最基本的HTML, CSS和JavaScript知识，受益匪浅。

但是在第五周开始讲Javascript的框架和相关应用如DOM, Ajax等概念时，我明显感受到难度加深，出现了看得懂文字，无法理解内容的情况，于是在知乎的相关问题中下载了两本大家推荐的入门书中文版，分别是《JavaScript DOM编程艺术》和《JavaScript语言精粹》。打算先开始读第一本书，并配合W3C过一遍CSS和JavaScript样式。

Fri July 20 2018

开始阅读《JavaScript DOM编程艺术》和《JavaScript语言精粹》。看完了第二章：JS语法和第三章：DOM，第四章图片库的案例研究看到childNodes属性。跟着书中的code过了一遍，加深印象，又是小小的进步。今天弄明白了DOM（Document Object Model)是什么，node，getElementById(), get/setAttribute的意义和用法，他们作为DOM的最基本要点，和数组的关系，一定是将来的常用选项，所以一定要多练多记。加油！

Mon July 23 2018

《JavaScript DOM编程艺术》结束第四至六章。第四章用DOM来解决了一个图片库的设计，用到了四个DOM的core function：getAttribute，setAttribute，getElementById和getElementByTagName。

第五章介绍了对于代码的优化，给我印象最深的是如何分离HTML，CSS和JavaScript的代码，来实现因为新增加的内容对于原有内容的最少修改。同时，assume nothing to make the structure robust.这大概就是高级程序员的不同所在了，不断地优化代码，使加载的速度变化，用户体验更好，才能在一众程序员中脱颖而出。

第六章就是对于图片库的优化了，今天草草看了一遍，有些地方仍不太理解，应该会在这两周重新温习。了解到了DOM可以在不同语言中使用，要同时熟悉HTML-DOM和DOM core替换属性的方法。

Tue July 24 2018

《JavaScript DOM编程艺术》看完了第七章动态创建标记和第八章的小案例：显示缩略语列表。了解到的是如何使用innerHTML属性，createElement,appendChild,CreateTextNode,insertBefore等方法来在JS中添加元素和文本。

Ajax仍然是一个难点，相关部分我没有理解，需要重温。
